Інфікційні захворювання.
Завдання проекту:Нориалізація та аналіз даних пандемій у SQL.
Етапи виконання:
1.Завантаження даних ( 7271 запісив у базі даних).
2.Нормалізація даних до 3-ї форми (створено схему pandemic) та довідник по країнах.
3.Аналіз дат,розраховано різницю між датаю запису та сьогодення у роках.
4.Віконано агрегацію даних для розрахунку макс,мін та середніх даніх по захворюваннямю
5.Виканано функцію розрахунку з 1996 року,яка повертає значення в роках від зазначеної даті.Ця функція внутрі кода-CalcalateYearDifference.
